## Deploy

Para fazer o deploy desse projeto rode

```cmd
  cd crud
```
```cmd
  npm install
```
```cmd
  npm start
```
```cmd
  cd ..
```
```cmd
  cd servidor
```
```cmd
  npm install
```
Arrume as variaveis de ambiente localizadas no arquivo .env.local para poder fazer a conexão com o banco de dados, retire o trecho .local do nome do arquivo.
```cmd
  npm run dev
```
